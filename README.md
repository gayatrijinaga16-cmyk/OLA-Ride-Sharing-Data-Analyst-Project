# OLA Ride-Sharing Data Analysis Dashboard

## Tools Used
- Power BI
- SQL (MySQL)

## Project Overview
Analyzed 100K+ OLA ride records to identify booking patterns, cancellation trends, revenue insights, and operational performance using Power BI and SQL.

## Key Insights
- Analyzed 100,000+ ride records
- Identified driver cancellations as a major operational issue
- Revenue and booking trend analysis
- City-wise and vehicle-type performance insights
- Built 5 custom KPIs using DAX
- Interactive dashboard with drill-down analysis
